Snowzone Adventure Land

"Snowzone Adventure Land" is an isometric puzzle game for Windows, Linux and Mac. The game allows a player to control up to four snowmen to guide them through more than 20 levels and reclaim their carrot noses. The project is developmed on an OpenGL rendering pipeline and uses an in-house physics engine to simulate character movement and projectile movement. The game includes a full-blown level editor capable of generating random terrain, placing AI enhanced enemies, creating trigger switches to alter game state, and other features.
